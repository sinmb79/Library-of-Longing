### A2. 블루오션 — 아무도 안 하는 것

기존 앰비언트 채널의 공통점: **"장소"를 보여주되 "사람"이 없다.** 벽난로는 있지만 그 옆에 앉아 있던 할머니가 없다. 카페는 있지만 내가 단골이던 카페가 아니다.

**그리움의 라이브러리가 공략할 블루오션:**

#### 카테고리 1: 기척의 앰비언스 (The Sound of Someone Being There)
- 옆방에서 누군가 요리하는 소리 + 멀리서 들리는 TV
- 다른 방에서 책장 넘기는 소리 + 가끔 들리는 머그컵 내려놓는 소리
- 누군가 집 안을 돌아다니는 발소리 + 먼 콧노래
- **핵심**: 혼자 사는 사람에게 "누군가 같은 집에 있다"는 감각

#### 카테고리 2: 시간 속 장소 (Places in Time)
- 1990년대 한국 시골집 여름 — 선풍기, 매미, 어머니 부엌 소리
- 1980년대 일본 여름 축제 — 풍경, 야타이, 먼 불꽃놀이
- 빅토리아 시대 영국 서재 — 벽난로, 만년필, 밖의 마차 소리
- 오스만 시대 시리아 정원 — 분수, 새소리, 바람
- **핵심**: 특정 시대 × 특정 문화의 교차점. 역사적 노스탤지어

#### 카테고리 3: 계절의 하루 (A Day in the Season)
- 봄비 내리는 시골집 아침 → 낮 → 저녁 (시간 흐름이 있는 앰비언스)
- 가을 단풍 아래 오후 → 해질녘 → 밤 (조명과 소리가 점진적 변화)
- **핵심**: 정적인 루프가 아닌, 느리게 흐르는 시간

#### 카테고리 4: 공동체의 배경음 (The Background of Belonging)
- 명절 부엌 — 찌개 끓는 소리, 어른들 대화, 아이들 뛰어다니는 소리
- 동네 골목 여름 저녁 — 먼 아이들, 매미, 저녁 뉴스 소리
- 시장 골목 — 호객 소리, 가위질, 먼 라디오
- **핵심**: "내가 속했던 세계"의 소리

#### 카테고리 5: 보편적 그리움의 지역화 (Universal Longing, Localized)

| 문화권 | 그리움의 장면 | 감각 요소 |
|--------|------------|-----------|
| **한국** | 할머니 집 마루, 여름 시골, 명절 부엌, 겨울 연탄난로 | 선풍기, 매미, 부침개, 연탄 타는 냄새의 시각화 |
| **일본** | 여름 축제, 다다미 방, 센토(목욕탕), 벚꽃 아래 | 풍경(ふうりん), 세미(蝉), 게타 소리, 물 흐르는 소리 |
| **미국/유럽** | 크리스마스 이브, 시골 농가, 여름 캠프, 할머니 부엌 | 벽난로, 오븐, 귀뚜라미, 먼 교회 종 |
| **동남아** | 스콜 내리는 처마 밑, 시장, 강가 | 열대우림, 스콜, 시장 웅성거림, 보트 엔진 |
| **중동** | 정원 분수, 아잔, 차 끓이는 소리 | 분수, 새, 바람, 멀리서 들리는 기도 |
| **라틴아메리카** | 아부엘라 부엌, 마을 광장, 열대 밤 | 라디오 볼레로, 요리 소리, 귀뚜라미, 먼 음악 |
| **인도** | 몬순, 차이왈라, 템플 벨 | 빗소리, 찻잔, 종소리, 도비가트 물소리 |

## Part B: 영상/오디오 구조 설계

### B1. 왜 AI가 오히려 유리한가

사람들이 원하는 것은 **실사가 아니라 기억의 질감**이다.

- 실사: 카메라 흔들림, 갑자기 차 소리, 해상도 불균일 → 환상 깨짐
- AI 생성: 태생적으로 **약간 비현실적** → 몽환적 질감 = 아련함에 최적
- 지브리 배경이 실사보다 더 그리운 이유: 완벽한 현실이 아니라 **정제된 감각**

**lo-fi 선호 연구 근거:**
PLOS One(2022) 질적 연구에서 ASMR 시청자들이 lo-fi(저음질) 사운드를 선호.
의도적으로 불완전한 음질이 "유사 기억(pseudo memory)"을 생성하여 노스탤지어를 유발.
→ AI 생성 특유의 약간의 불완전함이 오히려 강점

### B2. 영상 구조

#### 레이어 구조 (3-Layer Composition)

```
Layer 3: Foreground Detail (전경 디테일)
  - 장작 불꽃, 빗방울, 커튼 흔들림, 촛불 등
  - AI 생성 루프 5~10초 → 무한 반복

Layer 2: Mid-ground Scene (중경 장면)
  - 방 전체, 창밖 풍경, 마루 등
  - AI 생성 정적 이미지 + 미세 움직임(파랄랙스, 빛 변화)

Layer 1: Background Atmosphere (원경 분위기)
  - 하늘 변화, 먼 산, 안개 등
  - 매우 느린 변화 (10시간 동안 새벽→낮→저녁→밤)
```

#### 포맷 사양

| 항목 | 사양 |
|------|------|
| 해상도 | 4K (3840×2160) — TV/모니터 풀스크린 대응 |
| 프레임레이트 | 24fps (영화적 감성) 또는 30fps |
| 영상 길이 | 기본 3~5분 루프 → FFmpeg로 10시간 합성 |
| 루프 방식 | Seamless loop (시작=끝 프레임 일치) |
| 색감 | 따뜻한 톤, 약간의 필름 그레인, 비네팅 |
| 스타일 | 지브리풍 / 수채화풍 / 유화풍 (실사 X) |

#### 시간 흐름형 (고급)

정적 루프가 아닌, 느린 시간 변화가 있는 영상:
- 0:00~2:30 — 이른 아침 (푸른 빛, 새소리)
- 2:30~5:00 — 아침 (따뜻한 빛, 활기)
- 5:00~7:30 — 오후 (황금빛, 나른함)
- 7:30~10:00 — 저녁~밤 (주황→남색, 귀뚜라미)

→ 각 구간을 별도 AI 이미지로 생성 후 크로스페이드로 연결

### B3. 오디오 구조

#### 오디오 레이어 (4-Layer Sound Design)

```
Layer 4: Rare Events (희귀 이벤트) — 5~15분 간격
  - 먼 천둥, 새 한 마리 지저귐, 고양이 울음, 문 여닫는 소리
  - 예측 불가능한 요소 → "살아있는" 느낌 부여

Layer 3: Periodic Elements (주기적 요소) — 30초~2분 간격
  - 장작 탁탁, 빗물 세기 변화, 바람 강약
  - 단조로움 방지하면서도 예측 가능한 리듬

Layer 2: Continuous Base (연속 기반음)
  - 빗소리, 바람, 물소리, 선풍기 등
  - 일정한 배경음 유지

Layer 1: Sub-bass / Room Tone (공간음)
  - 방의 울림, 저주파 공기감
  - 헤드폰 사용 시 "공간 안에 있는" 감각 부여
```

#### 오디오 품질 기준

| 항목 | 사양 |
|------|------|
| 포맷 | 48kHz / 24bit 이상 |
| 스테레오 | 바이노럴 스테레오 (헤드폰 최적화) |
| 마스터링 | -14 LUFS (YouTube 표준) / 피크 방지 |
| 루프 이음새 | 크로스페이드 5~10초 |
| 총 길이 | 오디오는 영상과 독립적으로 30분+ 원본 제작 후 반복 |

#### 음악 사용 가이드

**원칙: 음악은 최소화하거나 없앤다.**

- 대부분의 성공적 앰비언트 채널은 음악 없음 (순수 환경음)
- 음악을 넣을 경우: 매우 느린 앰비언트 드론 / 피아노 단음 / 하프
- AI 앰비언트 음악 생성 도구: Mubert, Suno, 또는 로컬 AudioCraft
- **핵심**: 음악이 "들리면" 실패. 공기처럼 느껴져야 함

#### Phase 1: 프로토타입 1편 제작 (2~3일)

**선정 테마:** "할머니 집 마루, 여름 — 선풍기와 매미 소리"

```
Step 1: SDXL로 장면 이미지 생성
  Prompt: "korean traditional hanok maru wooden floor,
  old electric fan spinning, summer afternoon sunlight
  through paper door, watercolor ghibli style,
  warm nostalgic atmosphere, 4k"
  Negative: "realistic photo, modern, dark, gloomy"
  → 4~5장 생성 후 최고 품질 선택

Step 2: Wan2.2로 루프 애니메이션 생성
  → 같은 이미지를 First Frame & Last Frame에 입력
  → Motion prompt: "gentle fan spinning, light breeze
     moving curtain slightly, dust particles in sunlight"
  → 5~8초 루프 생성
  → RIFE로 프레임 보간 (24fps)

Step 3: FFmpeg로 10시간 합성
  ffmpeg -stream_loop 7200 -i loop_5sec.mp4 \
    -vf "noise=alls=15:allf=t,vignette" \
    -c:v libx264 -preset slow -crf 18 \
    -t 36000 output_video_10h.mp4

Step 4: 오디오 제작
  Layer 1: 방 공간음 (Room Tone) — Freesound.org
  Layer 2: 선풍기 연속음 — Freesound.org
  Layer 3: 매미 소리 (강약 변화) — AI 생성 or Freesound
  Layer 4: 가끔 들리는 풍경 소리, 먼 참새, 얼음 녹는 소리
  → Audacity에서 4레이어 믹싱 → 30분 원본
  → FFmpeg로 10시간 반복:
  ffmpeg -stream_loop 20 -i audio_30min.wav \
    -af "afade=t=out:st=35990:d=10" \
    -t 36000 output_audio_10h.wav

Step 5: 영상+오디오 머지
  ffmpeg -i output_video_10h.mp4 -i output_audio_10h.wav \
    -c:v copy -c:a aac -b:a 256k \
    -map 0:v:0 -map 1:a:0 \
    final_10h.mp4

Step 6: 썸네일 생성
  → ComfyUI로 같은 장면의 변형 이미지 생성
  → 텍스트 오버레이: "할머니 집 마루 | 여름 매미와 선풍기 | 10시간"
```
