**Goal:** Build the scene configuration contract and ambient audio mixer needed to produce the first prototype scene from a single YAML input.

**Architecture:** Keep the contract strict and reusable: `scenes/schema.yaml` documents and validates the scene shape, `scripts/scene_config.py` loads and normalizes YAML into a consistent structure, and `scripts/audio_mixer.py` consumes that normalized structure to render stereo ambient audio. Tests drive the behavior using generated WAV fixtures so the mixer is verified without external assets.

**Tech Stack:** Python 3.12, PyYAML, jsonschema, numpy, soundfile, pyloudnorm, pytest

def load_scene_config(path: Path) -> dict[str, Any]:

def mix_scene_audio(scene_path: Path, output_path: Path, duration_sec: int, seed: int) -> Path:
    scene = load_scene_config(scene_path)
    layers = render_layers(scene["audio"]["layers"], duration_sec, seed)
    mix = master_mix(layers, target_lufs=-14.0)
    sf.write(output_path, mix, 48000, subtype="PCM_24")
    return output_path
